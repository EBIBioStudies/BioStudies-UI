ArrayExpress to BioStudies API Migration Guide

We are in the process of migrating ArrayExpress to BioStudies as a project. As a result, there would be some changes in the API support for ArrayExpress. Until announced, the current version of the API will continue to work. For early access, here's a list of all API endpoints and their BioStudies equivalent. If you are using the ArrayExpress API to automate any part of your workflow, please take a look at the revised endpoints as well as the new schema of the responses. The endpoint has changed from https://www.ebi.ac.uk/arrayexpress/xml/v3 to https://www.ebi.ac.uk/biostudies/api/v1/arrayexpress. Please send all questions and feedback to biostudies@ebi.ac.uk.

Field name What is searched? ArrayExpress Example BioStudies URL
accession Experiment primary ArrayExpress or secondary (GEO, ENA, EGA etc) accession <endpoint>/experiments?accession=E-MEXP-31 <endpoint>/files/E-MEXP-31/E-MEXP-31.json
array Array design accession or name (wildcards supported) <endpoint>/files?array=A-AFFY-33 TBD
expdesign Experiment design type, related to the questions being addressed by the study, e.g. "time series design", "stimulus or stress design", "genetic modification design". Has EFO expansion. <endpoint>/files?expdesign=dose+response <endpoint>/search?experimental_design="dose response"
exptype Experiment type, related to the assay technology used. List of experiment types in ArrayExpress. Has EFO expansion. <endpoint>/experiments?exptype="RNA-seq of non coding RNA" <endpoint>/search?study_type="RNA-seq of non coding RNA"
ef or ev Experimental factor (also called experimental variable), the name of the main variable under study in an experiment. E.g. if the factor is "sex" in a human study, the researchers would be comparing between male and female samples, and "sex" is not merely an attribute the samples happen to have. Has EFO expansion. <endpoint>/experiments?ef="cell type" <endpoint>/search?experimental_factor="cell type"
efv or evv The value of an experimental factor. E.g. The values for "genotype" factor can be "wild type genotype", "p53-/-". Has EFO expansion. <endpoint>/experiments?efv=HeLa <endpoint>/search?experimental_factor_value=HeLa
sa Sample attribute values, e.g. "male", "liver". Has EFO expansion. <endpoint>/files?sa=fibroblast <endpoint>/search?source_characteristics=fibroblast
sac Sample attribute category that is defined in an experiment, e.g. "age", "cell type", "disease". Has EFO expansion. <endpoint>/files?sac=age <endpoint>/search?source_characteristics_value=age
species Species of the samples. Can use common name (e.g. "mouse") or binomial nomenclature/Latin names (e.g. "Mus musculus"). Has EFO expansion. <endpoint>/experiments?species="homo sapiens" <endpoint>/search?organism="homo sapien"
pmid PubMed identifier <endpoint>/experiments?pmid=16553887 <endpoint>/search?link_type=pmid&link_value=16553887
gxa Presence ("true") / absence ("false") of an ArrayExpress experiment in the Expression Atlas. <endpoint>/experiments?gxa=true <endpoint>/search?gxa=true
directsub If "true" only returns experiments directly submitted to ArrayExpress (i.e. not imported from GEO). For more information about how we import data from GEO see the GEO data help page. <endpoint>/experiments?directsub=true TBD
raw Experiment has raw data available. <endpoint>/experiments?raw=true <endpoint>/search?raw=true
processed Experiment has processed data available. <endpoint>/experiments?processed=true <endpoint>/search?processed=true
assaycount [x TO y] The number of of assays where x <= y and both values are between 0 and 99,999 (inclusive). To count excluding the values given use curly brackets e.g. assaycount={1 TO 5} will find experiments with 2-4 assays. Single numbers may also be given e.g. assaycount=10 will find experiments with 10 assays. <endpoint>/experiments?assaycount=[1 TO 5] TBD
samplecount [x TO y] The number of samples <endpoint>/experiments?samplecount=[1 TO 5] TBD
efcount [x TO y] The number of experimental factors https://www.ebi.ac.uk/arrayexpressxml/v3/experiments?efcount=[1 TO 5] TBD
sacount [x TO y] The number of sample attribute categories <endpoint>/experiments?sacount=[1 TO 5] TBD
miamescore [x TO y] The MIAME compliance score (maximum score is 5) <endpoint>/experiments?miamescore=[1 TO 5] TBD
minseqe [x TO y] The MINSEQE compliance score (maximum score is 5) <endpoint>/experiments?minseqescore=[1 TO 5] <endpoint>/search?minseq_score=[1 TO 5]
date The release date of the experiment. Format is [YYYY-MM-DD]. Wildcards supported. For example:
date=2009-12-01 will search for experiments released on 1st of Dec 2009.
date=2009* will search for experiments released in 2009.
date=[2008-01-01 2008-05-31] will search for experiments released between 1st of Jan and end of May 2008. 		<endpoint>/experiments?date=[2008-01-01 2008-05-31] <endpoint>/search?release_date=[2008-01-01 TO 2008-05-31]
